# Screenplay Writer / 剧本创作助手

![分类](https://img.shields.io/badge/分类-创意-green)

## 项目简介

影视剧本创作技能。从概念到完整剧本，支持电影、短剧、广告片等类型的剧本写作。

## 创作流程

1. 概念开发：核心创意和主题
2. 人物设定：角色性格、关系、弧光
3. 大纲构建：三幕式结构
4. 场景编写：对话和动作描写
5. 修改打磨：节奏、张力、连贯性

## 剧本格式

遵循标准剧本格式：
- 场景标题行
- 动作描述
- 角色名
- 对话
- 括号提示
