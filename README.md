# Screenplay Writer - 资深影视编剧与剧本医生

![OpenClaw Skill](https://docs.openclaw.ai/images/skill-badge.svg)

一个专业的影视剧本创作技能，将你的故事概念转化为符合好莱坞标准的文学剧本。

## 🎬 技能概述

作为拥有20年经验的资深电影编剧和剧本医生，这个技能精通：
- **好莱坞标准剧本格式**
- **视听语言心理学**  
- **蒙太奇叙事技巧**
- **人物弧光构建**

核心能力是将平淡的文字描述转化为具有强烈画面感、潜台词丰富、逻辑严密的文学剧本。

## 🚀 核心特性

### ✨ 创作原则
- **展示而非讲述**: 通过具体动作外化角色心理状态
- **视听思维**: 预设镜头存在，利用光影声音构建场景
- **潜台词优先**: 对话含蓄但张力十足，避免直白表达
- **道具叙事**: 关键道具承载象征意义，映射人物成长

### 📝 输出规范
- **标准剧本格式**: 场景标题、动作描述、角色名、台词
- **完全可视化**: 所有描述都可直接拍摄
- **自然转场**: 通过视觉匹配实现场景过渡
- **专业质量**: 符合电影工业标准

### 🚫 严格禁止
- 导演工作术语（"特写"、"8秒"、"镜头推进"等）
- 分镜表格或Excel格式
- 生硬转场（"画面切换至"）
- 过度依赖画外音解释剧情

## 📋 使用场景

### 适用情况
- 优化或重写现有影视剧本
- 将故事概念转化为标准文学剧本  
- 改进剧本的叙事结构和人物弧光
- 获得专业级别的剧本创作指导

### 触发示例
```
优化这个剧本：[剧本内容]
```
```
将这个故事写成专业剧本：[故事梗概]
```
```
重写这个场景，让它更有电影感：[场景描述]
```

## 📦 安装部署

### OpenClaw 集成
1. 克隆此仓库到你的 OpenClaw 工作区
2. 确保目录结构包含以下文件：
   - `SKILL.md` - 技能描述和使用指南
   - `_meta.json` - 技能元数据
   - `_skillhub_meta.json` - SkillHub 元数据

### 目录结构
```
screenplay-writer/
├── SKILL.md
├── _meta.json  
├── _skillhub_meta.json
└── README.md
```

### 依赖要求
- OpenClaw v2026.4.0 或更高版本
- 无需额外 Python 依赖

## 🎭 创作质量保证

### 光影运用
- 构建贯穿全片的视觉意象
- 利用自然光和人工光源反映角色心理
- 光影变化具有逻辑性和象征意义

### 人物弧光
- 心理逻辑链条严密
- 行为改变有充分内在动机  
- 通过对比展现成长转变

### 转场设计
- 视觉匹配实现自然过渡
- 声音先入或动作延续连接场景
- 避免生硬场景跳跃
