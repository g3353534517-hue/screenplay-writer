## 项目简介

专业的影视剧本创作技能，将故事概念转化为符合好莱坞标准的文学剧本。20年经验的资深编剧，精通好莱坞标准剧本格式、视听语言心理学、蒙太奇叙事技巧、人物弧光构建。核心能力是将平淡文字转化为画面感强、潜台词丰富、逻辑严密的剧本。
